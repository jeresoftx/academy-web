
## Concepto

Algoritmos no se presentan como recetas, sino como criterio para encontrar un
camino cuando el problema todavía es confuso. La escena debe mostrar el momento
en que una persona convierte puntos dispersos en una ruta verificable.

## Metáfora

Una mano dibuja o conecta nodos sobre una mesa de trabajo técnica. El grafo no
es decoración: representa búsqueda, decisiones, ventanas, rutas y costo.

## Historia Visual

1. El problema empieza como una constelación de nodos sin lectura inmediata.
2. El ingeniero identifica restricciones y conecta posibilidades.
3. Una ruta iluminada aparece como resultado de criterio, no de magia.

## Layout

- Desktop: texto a la izquierda, escena a la derecha.
- Mobile: código del curso arriba, trayectoria central y zona inferior segura.
- YouTube: sujeto grande, ruta muy clara, poco texto y contraste alto.

## Prompt Seed

Create editorial concept art for a software engineering course about algorithms.
Show a human hand drawing a luminous golden path through graph nodes on a dark
technical workbench. The image should communicate clarity emerging from a
scattered problem space: search, paths, sliding windows and complexity as visual
motifs, not readable UI. Warm amber light, black and ivory academy palette,
premium technical craftsmanship, clean negative space for external typography.
