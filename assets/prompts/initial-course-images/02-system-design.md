# System Design - prompt de imagen

## Guion visual

La imagen debe comunicar que System Design ensena a razonar sobre sistemas
reales: nodos, replicas, colas, latencia, limites y decisiones de capacidad.

Escena:

- ingeniero frente a monitores o tablero oscuro;
- diagramas flotantes de nodos, replicas, cola, base de datos y limites;
- lineas conectan componentes como arquitectura de sistema;
- ambiente serio, nocturno, de sala tecnica;
- no debe verse como dashboard corporativo generico, sino como pensamiento de
  arquitectura en accion.

Texto que ira fuera de la imagen, no dentro:

```text
S4
System Design
Nodos, replicas, latencia, colas y limites.
```

## Prompt para ChatGPT Images

```text
Create a 1600x900 horizontal 16:9 course cover illustration for Jeresoft Academy.

Subject: System Design for scalable software systems.

Scene: a focused software engineer in a dark technical workspace analyzing a system architecture. Around the engineer, show clean floating diagrams: service nodes, replicas, queue, database, cache, latency indicator, and capacity limits. Components should be connected by thin golden lines, like a system design diagram becoming alive.

Visual style: modern technical editorial illustration, sober engineering atmosphere, premium educational look, black dominant background with warm golden amber accents, ivory highlights, subtle depth, realistic but slightly stylized. It should feel like careful architecture thinking, not a generic SaaS dashboard.

Composition: leave clean safe space on the left and lower-left for website text overlays. Keep the engineer and system diagram as the main visual focus. The diagram should be understandable as architecture, but avoid readable product UI text.

Palette: #0a0907 black, #11100d dark panel, #ffd21a solar yellow, #d99a32 gold, #f0bd64 warm light, #f5efe6 ivory. Avoid blue/cyan dominance; if any cool light appears, keep it very subtle and secondary.

Rules: no logos, no brand names, no real company references, no generated headline text, no fake metrics as central content, no cyberpunk overload, no purple palette. Small technical labels like "queue", "replica", or "latency" are acceptable only if clean and minimal.

Output: one polished image, 1600x900, suitable as a course cover.
```

## Ajustes minimos en Canva

- Si aparecen textos raros en pantallas, cubrirlos o recortarlos.
- Reservar espacio para overlays del sitio.
