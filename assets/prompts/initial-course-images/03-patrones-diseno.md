# Patrones de diseno - prompt de imagen

## Guion visual

La imagen debe comunicar patrones de diseno como oficio: piezas, composicion,
interfaces, estructura y soluciones reutilizables que ayudan a construir mejor.

Escena:

- mesa tecnica o laptop oscuro con codigo desenfocado;
- una mano sostiene o ensambla una pequena estructura modular;
- piezas tipo puente, bloques, interfaces o componentes encajan entre si;
- la composicion sugiere "arquitectura pequena" y criterio de diseno;
- atmosfera negra/dorada/marfil, elegante y sobria.

Texto que ira fuera de la imagen, no dentro:

```text
DP
Patrones de diseno
Piezas, composicion, interfaces y estructura.
```

## Prompt para ChatGPT Images

```text
Create a 1600x900 horizontal 16:9 course cover illustration for Jeresoft Academy.

Subject: Design Patterns as software craftsmanship, composition, interfaces, and reusable structure.

Scene: a dark elegant engineering desk with a laptop or editor in the background, softly out of focus. In the foreground, a human hand is assembling a small modular structure made of clean geometric pieces, like a bridge, framework, or component system. The pieces should suggest composition, interfaces, contracts, and reusable patterns in software design.

Visual style: modern technical editorial illustration, black dominant background, warm golden amber light, ivory highlights, premium but sober, practical engineering craft. It should feel like building a precise structure, not like abstract corporate art.

Composition: leave clean safe space on the left and lower-left for website text overlays. Main focus should be the hand and modular structure. Background code can be present but should not contain readable generated text.

Palette: #0a0907 black, #11100d dark panel, #ffd21a solar yellow, #d99a32 gold, #f0bd64 warm light, #f5efe6 ivory.

Rules: no logos, no brand names, no real company references, no generated headline text, no random puzzle clipart, no cartoon style, no purple/blue dominance. The modular structure should look elegant and technical, with clear geometry and warm highlights.

Output: one polished image, 1600x900, suitable as a course cover.
```

- Mantener la estructura modular como foco.
- No convertirlo en iconografia de rompecabezas generica.
