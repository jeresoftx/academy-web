# Algoritmos - prompt de imagen

## Guion visual

La imagen debe comunicar que algoritmos son la base del criterio tecnico:
ver patrones invisibles, encontrar caminos y convertir un problema confuso en
una ruta clara.

Escena:

- fondo oscuro elegante con grilla tecnica sutil;
- una mano dibuja o conecta nodos luminosos sobre un plano digital;
- nodos y aristas forman un grafo/camino;
- se sugieren busqueda, ventanas y complejidad con pequenos diagramas;
- atmosfera negra, dorada y ambar, con luz calida;
- debe sentirse como ingenieria artesanal, no como fantasia futurista.

Texto que ira fuera de la imagen, no dentro:

```text
S1
Algoritmos
Grafos, caminos, ventanas, busqueda y complejidad.
```

## Prompt para ChatGPT Images

```text
Create a 1600x900 horizontal 16:9 course cover illustration for Jeresoft Academy.

Subject: Algorithms as the foundation of software engineering thinking.

Scene: a dark elegant technical workspace with a subtle engineering grid. A human hand is drawing or connecting glowing golden nodes on a digital blueprint. The nodes and edges form a clear graph path. Add subtle visual hints of binary search, sliding window, paths, and complexity analysis as small abstract technical marks, not readable UI text.

Visual style: modern technical editorial illustration, premium but not luxury advertising, black dominant background, warm golden amber light, ivory highlights, sharp contrast, subtle terminal/editor atmosphere, software craftsmanship, clean composition.

Composition: leave clean safe space on the left and lower-left for website text overlays. Main visual focus should be the hand, the graph nodes, and the illuminated path. Avoid clutter.

Palette: #0a0907 black, #11100d dark panel, #ffd21a solar yellow, #d99a32 gold, #f0bd64 warm light, #f5efe6 ivory.

Rules: no logos, no brand names, no real company references, no generated headline text, no fake UI with unreadable text, no cyberpunk neon overload, no blue/purple palette. Small abstract monospace labels are acceptable only if they are decorative and not central.

Output: one polished image, 1600x900, suitable as a course cover.
```
